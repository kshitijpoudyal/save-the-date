# Kshitij & Asmita - Save the Date 💒

A beautiful wedding save-the-date website built with React and Vite.

## Features

- Elegant hero section with countdown timer
- Wedding details and venue information
- Photo gallery
- Fully responsive design
- Smooth animations with Framer Motion

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone <your-repo-url>
cd kshitij-asmita-save-the-date

# Install dependencies
npm install

# Start development server
npm run dev
```

The site will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The build output will be in the `dist/` folder.

## Deployment to Vercel

1. Push your code to GitHub
2. Import the repository in [Vercel](https://vercel.com)
3. Deploy with default settings (Vite is auto-detected)

No environment variables are required.

## Tech Stack

- **React 18** - UI framework
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **React Router** - Navigation
- **Lucide React** - Icons
